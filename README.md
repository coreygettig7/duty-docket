# **Duty Docket**
Duty Docket is a place where families can divide and conquer their household duties.    

### **User Story**
As a USER 
I **WANT** to have **ONE** place **WHERE** I can **CREATE** duties for family members   
I **WANT** to be able to **ASSIGN** duties   
I **WANT** to be able to **ADD**, **EDIT**, or **REMOVE** a **DUTY**   
I **WANT** to have **ONE** place where I can **MANAGE** household **DUTIES**   

## **Installation**
  1. Clone the repository from [GitHub](https://github.com/coreygettig7/duty-docket.git)   
  2. Install Dependencies with `npm i`   
  3. Run `npm start` from your command line to turn the server on an make the app live on your localhost     
  4. Check the console for any issues, check pacakage.json for any missing dependencies   
  5. Deploy your app to [Heroku](https://devcenter.heroku.com/articles/deploying-nodejs)   
 
## **Deployment**
This fantastic app is deployed with [AtlasDB](https://www.mongodb.com/developer/how-to/use-atlas-on-heroku/) and [Heroku](https://devcenter.heroku.com/articles/deploying-nodejs), click [HERE](https://dutydocket.herokuapp.com/) to start your duty adventure.
